Getting Started with Ruby 

            
Get a Binary Value

            
                require ‘net/http’ 
                require ‘uri’

                args = { :stat => stat_name, :email => account_email, :count => count } 
                resp = Net::HTTP.post_form(URI.parse("http://localhost:8081/api/binary"), args)
            
        
            
Convert a Binary Value to Decimal

            
                require ‘net/http’ 
                require ‘uri’

                args = { :stat => stat_name, :email => account_email, :count => count } 
                resp = Net::HTTP.post_form(URI.parse("http://localhost:8081/api/binary/1010/convert/decimal"), args)
